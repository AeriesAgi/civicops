# Integration Readiness

CivicOps is pilot-ready architecture, not an official municipal deployment.

Primary channels:
1. Mobile/PWA citizen app
2. Web reporting portal
3. Public reference/status lookup
4. Area alerts/weather notices
5. Optional WhatsApp connector-ready integration

Live integration path:
- Configure Gemini credentials and enable event-triggered actions.
- Add municipal/NGO/public-utility data-sharing agreements.
- Configure WhatsApp Business only if a pilot requires it.
- Add storage, identity, notification and ERP/GIS connectors through environment-gated adapters.
- Keep human review before dispatch or public alert publication.
